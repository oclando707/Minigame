接口设计

# scripts/interactable.gd
extends Area2D

## 当玩家按下交互键时触发
signal interacted

## 是否显示提示图标（按F交互）
@export var show_prompt: bool = true

## 是否可以多次交互（false = 一次性）
@export var repeatable: bool = true

# 内部状态（与原代码行为一致）
var player_in_range: bool = false
var interaction_locked: bool = false

# 公开方法
func unlock_interaction() → void

与原有代码的兼容对照

原行为 新实现
原脚本 _on_body_entered：显示提示图标 ✅ 完全相同逻辑
原脚本 _on_body_exited：隐藏提示图标 ✅ 完全相同逻辑
原脚本 _process：检测 interact 输入 ✅ 完全相同逻辑
原脚本 interaction_locked = true 阻止重复触发 ✅ 完全相同
原脚本 unlock_interaction() 允许再次交互 ✅ 完全相同
新增：@export repeatable 扩展 🆕 可配置一次性/可重复交互
新：收音机迁移到这个系统 🆕 radio 获得重复交互能力
